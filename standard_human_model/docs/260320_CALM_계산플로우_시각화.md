작성일: 2026-03-20

## 1. 전체 파이프라인 개요

```
외부 입력 (IsaacGym 시뮬레이터)
  ├─ dof_pos      (num_envs, 69)   ← 현재 관절 각도 [rad]
  ├─ dof_vel      (num_envs, 69)   ← 현재 관절 각속도 [rad/s]
  ├─ descending_cmd (num_envs, 20) ← 상위 제어기의 근육 명령 [0,1]
  └─ contact_forces (num_envs, 4)  ← 발바닥 접촉력 [N]
         │
         ▼
  ╔══════════════════════════════════════════════════════════╗
  ║                   HumanBody.compute_torques()            ║
  ║                                                          ║
  ║  Step 1: Muscle Kinematics      (moment_arm.py)          ║
  ║  Step 2: Reflex Layer           (reflex_controller.py)   ║
  ║  Step 3: Activation Dynamics    (activation_dynamics.py) ║
  ║  Step 4: Force Generation       (muscle_model.py)        ║
  ║  Step 5: Torque Mapping         (moment_arm.py)          ║
  ║  Step 6: Ligament Forces        (ligament_model.py)      ║
  ╚══════════════════════════════════════════════════════════╝
         │
         ▼
  tau_total (num_envs, 69) → IsaacGym apply_forces()
```

## 2. 단계별 상세 플로우

```
┌──────────────────────────────────────────────────────────────────────┐
│  INPUT                                                               │
│  dof_pos (B,69)  dof_vel (B,69)  descending_cmd (B,20)              │
└────────────┬─────────────┬─────────────┬────────────────────────────┘
             │             │             │
             ▼             ▼             │
┌─────────────────────────────────────┐ │
│  STEP 1: MUSCLE KINEMATICS          │ │
│  moment_arm.py                      │ │
│                                     │ │
│  R (20, 69) — 상수 행렬              │ │
│  (근육 × DOF, 현재 각도 독립)         │ │
│                                     │ │
│  l_muscle = l_slack - R @ dof_pos   │ │
│             (B, 20)   (B, 20)       │ │
│                                     │ │
│  v_muscle = -(R @ dof_vel)          │ │
│             (B, 20)   (B, 20)       │ │
│                                     │ │
│  💡 Bi-articular: hamstrings R에    │ │
│     Hip(-0.06)와 Knee(+0.03) 모두   │ │
│     비영값 → 두 관절 동시 커플링     │ │
└──────┬───────────────────┬──────────┘ │
       │ l_muscle, v_muscle│            │
       │                   │            ▼
       │            ┌──────┴────────────────────────────────────┐
       │            │  STEP 2: REFLEX LAYER                      │
       │            │  reflex_controller.py                      │
       │            │                                            │
       │            │  ① 신장 반사 (Stretch Reflex) — 지연 있음  │
       │            │     if v_muscle > threshold:              │
       │            │       Δa += gain × (v - threshold)        │
       │            │     [건강인: gain=1.0 / 경직: gain=8.0]   │
       │            │                                            │
       │            │  ② GTO 억제 (골지건 반사)                  │
       │            │     if F/f_max > gto_threshold:           │
       │            │       Δa -= gto_gain × excess             │
       │            │     [과부하 시 근육 보호]                  │
       │            │                                            │
       │            │  ③ 상반 억제 (Reciprocal Inhibition)       │
       │            │     quadriceps ↔ hamstrings               │
       │            │     tibialis_ant ↔ soleus/gastrocnemius    │
       │            │     → 주동근 강 → 길항근 억제              │
       │            │                                            │
       │            │  a_cmd = clamp(descending_cmd + Δa, 0,1)  │
       │            │          (B, 20)                          │
       │            └──────────────────┬─────────────────────────┘
       │                               │ a_cmd (B,20)
       │                               ▼
       │            ┌──────────────────────────────────────────┐
       │            │  STEP 3: ACTIVATION DYNAMICS             │
       │            │  activation_dynamics.py                  │
       │            │                                          │
       │            │  1차 ODE (근육별 독립):                   │
       │            │  if a_cmd > a_prev:                      │
       │            │    da = (a_cmd - a) / tau_act  [빠름]    │
       │            │    tau_act: 15~20ms (건강) / 50ms (파킨슨) │
       │            │  else:                                   │
       │            │    da = (a_cmd - a) / tau_deact [느림]   │
       │            │    tau_deact: 60~80ms / 200ms (파킨슨)   │
       │            │                                          │
       │            │  a(t+dt) = clamp(a + da×dt, 0, 1)       │
       │            │  activation (B, 20)                      │
       │            └──────────────────┬───────────────────────┘
       │                               │ activation (B,20)
       ▼                               ▼
┌──────────────────────────────────────────────────────────────┐
│  STEP 4: FORCE GENERATION                                     │
│  muscle_model.py  — Hill-type                                 │
│                                                              │
│  l_norm = l_muscle / l_opt        (B, 20)                    │
│  v_norm = v_muscle / (v_max × l_opt) (B, 20)                │
│                                                              │
│  F_active  = a × f_max × f_FL(l_norm) × f_FV(v_norm) × cos(α) │
│              ──── ────────────────────────────────────────── │
│              [0,1]  근육별 상수    Gaussian  Hill곡선  우각 보정 │
│                                                              │
│  f_FL(l) = exp(-((l-1)/σ)²)   ← 피크: l_norm=1.0           │
│  f_FV(v) = {                                                 │
│     단축(v<0):  (1 - v/v_max) / (1 + v/(c×v_max))  [약해짐] │
│     신장(v>0):  1.5 - 0.5×exp(-v)                  [강해짐] │
│  }                                                           │
│                                                              │
│  F_passive = f_max × k_pe × (exp(l_norm/ε₀) - 1)   [l>1일 때] │
│  F_damping = damping × f_max × v_muscle                     │
│                                                              │
│  F_total = clamp(F_active + F_passive + F_damping, 0, ∞)    │
│            (B, 20)                                          │
└──────────────────────────────┬───────────────────────────────┘
                               │ F_muscle (B,20)
                               ▼
┌──────────────────────────────────────────────────────────────┐
│  STEP 5: TORQUE MAPPING                                       │
│  moment_arm.py                                               │
│                                                              │
│  tau_muscle = F_muscle @ R          (matrix multiply)        │
│               (B,20) × (20,69) = (B,69)                     │
│                                                              │
│  예시: hamstrings_L 단독 활성화 시                            │
│    τ_Hip_x  += F_ham × (-0.06)  ← 고관절 신전 토크           │
│    τ_Knee_x += F_ham × (+0.03)  ← 무릎 굴곡 토크            │
│    [Bi-articular coupling 자동 처리]                         │
└──────────────────────────────┬───────────────────────────────┘
                               │ tau_muscle (B,69)
                               ▼
┌──────────────────────────────────────────────────────────────┐
│  STEP 6: LIGAMENT FORCES                                      │
│  ligament_model.py                                           │
│                                                              │
│  excess_upper = max(0, q - q_soft_upper)                     │
│  excess_lower = max(0, q_soft_lower - q)                     │
│                                                              │
│  tau_lig = -k_lig × (exp(α×excess_upper) - 1)               │
│           +k_lig × (exp(α×excess_lower) - 1)                │
│           -b_damp × dof_vel   [경계 구역 내에서만]            │
│                                                              │
│  [soft_limit_margin = 85%: 가동범위의 85% 지점부터 발동]       │
└──────────────────────────────┬───────────────────────────────┘
                               │ tau_ligament (B,69)
                               ▼
┌──────────────────────────────────────────────────────────────┐
│  OUTPUT                                                      │
│  tau_total = tau_muscle + tau_ligament  (B, 69)              │
│  → IsaacGym: gym.set_dof_actuation_force_tensor()            │
└──────────────────────────────────────────────────────────────┘
```

## 4. 환자 파라미터가 개입하는 위치

```
descending_cmd (B,20)
       │
       │──────────────────────────── [환자 프로파일 YAML]
       │                                      │
       ▼                                      │
  STEP 2: Reflex Layer ◄──────── stretch_gain  (경직: 8.0 / 건강: 1.0)
       │                          gto_gain     (경직 시 줄임)
       │                          reciprocal_gain (파킨슨: 줄임 → 공동수축)
       ▼
  STEP 3: Activation Dynamics ◄── tau_act      (파킨슨: 0.050s / 건강: 0.015s)
       │                          tau_deact    (파킨슨: 0.200s / 건강: 0.060s)
       ▼
  STEP 4: Force Generation ◄───── f_max        (편마비 환측: 40~60% 감소)
       │                          l_opt        (⚠️ 현재 1.0 무차원 — 버그!)
       │                          v_max        (파킨슨: 약간 감소)
       │                          pennation    (불변)
       ▼
  STEP 5: Torque Mapping ◄──────── R 행렬       (현재 상수, 각도 의존성 미구현)
       │
       ▼
  STEP 6: Ligament Model ◄──────── k_lig        (구축: 증가)
                                   soft_limit_margin (구축: 줄어듦)
                                   alpha        (구축: 증가)
```

## 5. Bi-articular 커플링 상세

```
근육 이름         관여 관절 (R 행렬 비영값)         기능
──────────────────────────────────────────────────────────────────
hamstrings_L      L_Hip (-0.06) + L_Knee (+0.03)   고관절 신전 + 무릎 굴곡
rectus_femoris_L  L_Hip (+0.04) + L_Knee (-0.04)   고관절 굴곡 + 무릎 신전
gastrocnemius_L   L_Ankle (-0.05) + L_Knee (+0.02) 발목 저측굴곡 + 무릎 굴곡

에너지 전달 예시 (Bobbert 1986):
  gastrocnemius 활성화
       │
       ├─ 무릎에서 에너지 흡수 (eccentric at knee)
       └─ 발목에서 에너지 생성 (concentric at ankle, push-off)
       → 무릎 → 발목 에너지 전달 (bi-articular 핵심 역할)
```

## 6. 시뮬레이터 연동 루프

```
┌─────────────────────────────────────────────────────────┐
│                  IsaacGym 시뮬 루프 (dt = 1/120 s)       │
│                                                         │
│  ┌─ RL Policy (상위 제어기) ────────────────────────┐   │
│  │  obs = [dof_pos, dof_vel, contact_forces, ...]  │   │
│  │  descending_cmd = policy.act(obs)  (B, 20)      │   │
│  └────────────────────────┬────────────────────────┘   │
│                           │                             │
│  ┌─ CALM Pipeline ────────▼────────────────────────┐   │
│  │  tau_total = human_body.compute_torques(        │   │
│  │      dof_pos, dof_vel,                          │   │
│  │      descending_cmd, dt, sim_time,              │   │
│  │      contact_forces                             │   │
│  │  )                                              │   │
│  └────────────────────────┬────────────────────────┘   │
│                           │ tau_total (B, 69)            │
│  ┌─ PhysX Engine ─────────▼────────────────────────┐   │
│  │  gym.set_dof_actuation_force_tensor(tau_total)  │   │
│  │  gym.simulate()                                 │   │
│  │  gym.fetch_results()                            │   │
│  │  → 새 dof_pos, dof_vel, contact_forces 생성     │   │
│  └─────────────────────────────────────────────────┘   │
│                                                         │
│  반복 ─────────────────────────────────────────→        │
└─────────────────────────────────────────────────────────┘
```

## 7. 현재 알려진 버그 및 미구현 항목

```
위치                   문제                          영향
──────────────────────────────────────────────────────────────────────
healthy_baseline.yaml  l_opt: 1.0 (무차원)           l_norm ≈ 0.3 → F_active 최대의 30%
                       실제 값 필요 (m 단위)           ← 최우선 수정 필요!

moment_arm.py          R 행렬이 상수                  무릎 굴곡 0→90°에서 실제
                       (각도 의존성 없음)              moment arm ±30% 변화 무시

muscle_model.py        Elastic Tendon (SE) 없음       아킬레스건 에너지 저장 미반영
                       rigid tendon 가정               → push-off 에너지 과소평가

reflex_controller.py   stretch reflex 지연이           delay_steps = 0 (즉각 반사)
                       현재 미구현 상태                생리학적으로는 25~60ms 지연
```
