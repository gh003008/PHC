# L2 프로파일 파라미터 조정 근거

작성일시: 2026-03-23 (일)
관련 테스트: I03 (프로파일 분화), I04 (Stretch Reflex 차등 저항)
관련 파일: `standard_human_model/validation/02_isaacgym_integration/run_validation.py`

---

## 1. 문제 상황

BUG-01 수정(l_opt 단위 정상화) 후 L2 검증을 재실행한 결과,
I01/I02는 PASS했으나 **I03/I04가 FAIL**.

```
I03: S=22.3° > H=9.9° > F=6.9°
     → S-H=12.4° ✓,  H-F=3.0° ✗ (기준: >5°)

I04: |H|=9.26 > |S|=7.37 > |F|=1.51
     → 순서 역전 (Healthy > Spastic) ✗
```

---

## 2. 근본 원인: BUG-01 수정의 연쇄 효과

`l_opt`가 1.0(무차원) → 실제 미터값(0.044~0.142m)으로 수정되면서
정규화 섬유 길이(l_norm)가 정상 범위로 복귀:

```
l_norm = l_ce / l_opt

수정 전: ~0.3m / 1.0 = 0.3  →  f_FL(0.3) ≈ 0.30  (능동력 최대치의 30%)
수정 후: ~0.3m / 0.3 = 1.0  →  f_FL(1.0) = 1.00  (능동력 최대치의 100%)
```

Healthy 기준선 근육력이 약 3배 증가.
기존 프로파일 파라미터는 "약한 근육" 기준으로 튜닝된 것이므로 재조정 필요.

---

## 3. I03 FAIL 분석

합격 기준: `Spastic > Healthy > Flaccid` 순서 + 각 간격 > 5°

- S-H 간격 12.4°는 충분
- **H-F 간격 3.0°가 부족** (기준 5°)

원인: Flaccid의 `f_max_scale=0.05`는 이전에는 `0.05 × 약한 힘 ≈ 0`이었으나,
수정 후 `0.05 × (3배 강한 힘)` = 여전히 유의미한 힘 → Healthy와 충분히 차별화되지 않음.

---

## 4. I04 FAIL 분석

합격 기준: `|Spastic| > |Healthy| > |Flaccid|` (초기 20step 평균 bio-torque 절댓값)

**순서가 역전**: Healthy(9.26 Nm) > Spastic(7.37 Nm)

원인: 근육 damping 토크 = `damping × f_max × v_norm`.
Healthy의 정상화된 f_max가 크게 증가하면서 기본 damping 토크만으로도 상당한 값.
Spastic의 `stretch_gain=8.0`으로 추가되는 reflex 토크가 이 기본 damping을 압도하지 못함.

즉, 기존 8배 게인은 "약한 근육 × 8 = 중간"이었는데,
수정 후 "강한 근육 × 1(Healthy damping)"이 이미 높아서 역전 발생.

---

## 5. 조정 내용 및 근거

### 5-1. Spastic (경직) 프로파일

| 파라미터 | 이전 | 조정 | 근거 |
|---------|------|------|------|
| stretch_gain | 8.0 | **15.0** | Lance 1980: 경직 환자의 신장 반사 게인은 정상의 10~20배. 8배는 경증(mild), 15배는 중등도(moderate)에 해당 |
| stretch_threshold | 0.02 | **0.01** | 경직의 핵심 기전인 velocity-dependent resistance: 더 낮은 속도에서도 반사 발동. Thilmann 1991 참고 |
| k_lig (인대 강성) | 200 | **300** | 만성 경직에서 결합조직 섬유화(fibrosis) + 관절 구축(contracture)으로 수동 강성 증가 |
| damping (인대 감쇠) | 25 | **40** | 동일 근거: 경직 관절의 조직 점탄성(viscoelasticity) 증가 |
| alpha (soft-limit 경사) | 15 | **18** | ROM 끝에서 더 급격한 저항. 관절 구축의 특성 |
| damping_scale (근육 점성) | 3.0 | **5.0** | Mirbagheri 2001: 경직 근육의 고유 점성(intrinsic viscosity)은 정상의 3~7배. 5배는 중등도 |

### 5-2. Flaccid (이완) 프로파일

| 파라미터 | 이전 | 조정 | 근거 |
|---------|------|------|------|
| f_max_scale | 0.05 | **0.005** | BUG-01 수정으로 기준 근력 3배 증가 → 이전 0.05의 실효값=0.15. 중증 SCI(MRC grade 0~1)는 수의적 근력 거의 0 → 0.005로 하향 |
| k_lig (인대 강성) | 5.0 | **3.0** | 탈신경(denervation) + 근위축(atrophy) → 수동 조직 강성 감소 |
| damping (인대 감쇠) | 0.5 | **0.2** | 동일 근거: 위축된 조직의 점성 저하 |
| alpha (soft-limit 경사) | 5.0 | **3.0** | 이완된 관절의 soft-limit 경사 완화 |
| damping_scale (근육 점성) | 0.1 | **0.05** | 근위축으로 고유 점성 감소. 탈신경 근육은 정상 점성의 5~10% 수준 |

### 5-3. Healthy (정상) — 변경 없음

기준선 자체가 BUG-01 수정으로 정상화되었으므로 조정 불필요.
stretch_gain=1.0, stretch_threshold=0.1 등 기본값 유지.

## 7. 핵심 교훈

1. **하위 버그 수정은 상위 검증 파라미터에 연쇄적 영향을 미침**.
   BUG-01(l_opt 단위) 수정 → 근육력 3배 → 프로파일 파라미터 재조정 필요.
   이는 검증 피라미드의 아래 레벨(L0)이 확실해야 위 레벨(L2)이 의미 있다는 원칙을 확인.

2. **파라미터 조정은 임상 문헌 범위 내에서 수행**.
   stretch_gain 15배(Lance 1980 범위 10~20), damping_scale 5배(Mirbagheri 2001 범위 3~7) 등
   모든 값이 문헌 보고 범위의 중간값에 해당.

3. **Flaccid는 "거의 0"이 핵심**.
   f_max_scale 0.05 → 0.005는 큰 변화처럼 보이지만,
   중증 SCI 환자(MRC grade 0~1)의 수의적 근력이 실질적으로 0에 가깝다는 임상 현실을 반영.
