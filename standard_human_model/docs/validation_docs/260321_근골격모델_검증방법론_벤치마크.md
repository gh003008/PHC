# 근골격 모델 검증 방법론 — MyoSuite / OpenSim 벤치마크 조사

작성일: 2026-03-21

## 배경 및 목적

CALM 검증 피라미드(ver2)의 L4 기준을 설계하기 전에,
기존 대표 프레임워크(MyoSuite, OpenSim)가 근골격 모델을
어떤 기준으로 검증했는지 조사하였다.
사용자의 직관적 기준(관절별 전수 / 근육별 전수 / 통합 데이터 플로우)이
실제 문헌의 계층 구조와 얼마나 일치하는지 확인하는 것이 목적이다.

## 1. MyoSuite 검증 방법 (Caggiano et al., 2022)

### 핵심 철학

MyoSuite는 **OpenSim → MuJoCo 변환 정확도** 검증을 수행한다.
원본 OpenSim 모델 자체의 생리학적 타당성은 이미 검증된 것으로 전제하고,
변환된 MuJoCo 모델이 원본을 얼마나 충실히 재현하는지만 검증한다.

### 3단계 검증 (Vlt1~Vlt3)

| 단계 | 검증 대상 | 정량 기준 |
|------|---------|---------|
| Vlt1 | Forward Kinematics (관절 기하, 마커 위치) | 시각적 수렴 확인 (수치 기준 없음) |
| Vlt2 | Moment Arm 곡선 (전체 ROM × 전체 근육) | Elbow: RMS 0.044±0.09% / Hand: RMS 0.38±0.57% |
| Vlt3 | Force-Length-Activation 관계 | Elbow: 2.2±1.4% Fmax / Hand: 4.1±2.0% Fmax |

MyoConverter 자동 파이프라인 합격 기준:
- Pearson r > 0.7
- 절대 RMSE < 1 mm (moment arm)
- normalized RMSE < 0.4

### MyoSuite가 검증하지 않은 항목

- 독립적 ROM 범위 (cadaver 대비)
- 모델 레벨 EMG 매칭
- 통합 보행 키네마틱 vs normative data
- GRF (지면반력) 재현성

→ 이 항목들은 "task 레벨 연구"로 분류하여 모델 레벨에서는 수행하지 않음.

## 2. OpenSim 검증 방법

### 2-1. Hicks et al. 2015 — "Is My Model Good Enough?"

OpenSim의 가장 공식적인 검증 가이드라인. 핵심 기준:

**Multibody Dynamics 수준:**
- 관절 각도/모멘트: 비교 문헌값의 **±2 SD** 이내
- Residual force: 측정된 net external force의 **≤ 5% (peak 및 RMS)**
- Residual moment: COM 높이 × net external force의 **≤ 1%**

**근골격 기하 수준:**
- Moment arm: 실험 데이터의 **±2 SD** 이내 (전체 ROM)
- Net joint moment: 실험값의 **±2 SD** 이내

**근육-건 동역학 수준:**
- 활성화 최대 오차: 9% / 비활성화 최대 오차: 16% (mean absolute force error)
- FL / FV / tendon 곡선: C2 연속 + 음의 강성 구간 없음 필수

**민감도 분석 우선순위 (필수 수행):**
1. Tendon slack length (가장 민감, tendon/fiber > 3인 근육에서 치명적)
2. Optimal fiber length
3. Maximum isometric force
4. Pennation angle (상대적으로 덜 민감)

### 2-2. Rajagopal et al. 2016 — Full-Body Musculoskeletal Model

MyoLeg의 기반 모델. 가장 구체적인 통합 보행 검증 수치.

**관절 모멘트 RMSE (통합 검증의 핵심 지표):**

| 관절 | 보행 오차 | 달리기 오차 |
|------|---------|---------|
| Hip | < 2% peak moment | < 3% peak |
| Knee | < 2% peak moment | < 3% peak |
| Ankle | < 2% peak moment | < 3% peak |

**운동학 오차:**
- 보행: 회전 < 1° RMSE, 골반 변위 < 0.1 cm
- 달리기: 회전 < 4° RMSE, 골반 변위 < 0.2 cm

**EMG 비교:**
- 정량 기준 없음. 주요 특징(타이밍, 피크 패턴)의 **정성적 일치**만 확인.

**유효 ROM (이 범위 내에서만 모델 유효):**
- 발목: 40° 저측굴곡 ~ 30° 배측굴곡
- 무릎: 0° ~ 120° 굴곡
- 고관절: 굴곡/신전 -30°~120°, 내외전 -30°~50°

## 3. 분야 전체 공식 벤치마크 현황

### Springer 2025 — Skeletal Muscle Models Benchmark

5개 벤치마크 케이스로 근수축 동역학 검증을 단계화:
- Case 1~3: 단일 근육 FL/FV 동역학
- Case 4~5: 다관절 시스템 통합

논문이 명시한 한계:
> "표준 참조 데이터가 없어 새 모델의 계산적 검증이 어렵다."
→ 통합 보행 검증(Kin+GRF+EMG 동시)의 공식 벤치마크는 **2025년 현재도 없음**.

### 통합 보행 검증에서 관행적으로 사용되는 기준

- 수직 GRF correlation: ρ > 0.94, normalized RMSE ≤ 5.3%
- 관절 모멘트 correlation 분류:
  - weak ≤ 0.35 / moderate 0.35~0.67 / strong 0.67~0.90 / excellent > 0.90
- EMG: KINESIS 2025 — Pearson correlation, 9명 × 9개 하지 근육

## 4. 항목별 비교 요약

| 검증 항목 | MyoSuite | OpenSim | 공식 벤치마크 |
|---------|---------|---------|-----------|
| 관절 ROM | 설계 범위 명시만 (검증 없음) | 사용 범위 내 < 1~4° RMSE | 없음 |
| 관절 토크 범위 | 미검증 | RMSE < 2~3% peak | 없음 |
| Force-Length 곡선 | < 4.1% Fmax RMS | C2 연속 + 9% MAE | Springer 2025 |
| Force-Velocity 곡선 | < 4.1% Fmax RMS | C2 연속 | Springer 2025 |
| Moment arm | < 0.38% RMS, r > 0.7 | ±2 SD (실험) | 부분적 |

## 5. 사용자 직관과 문헌의 대응 관계

사용자가 제시한 3단계 기준이 실제 프레임워크 검증 계층과 정확히 매핑된다.

```
사용자 직관                    →   문헌 대응

① 각 관절별 전수               →   Vlt1 (기하) + 관절 모멘트 RMSE (Rajagopal)
② 각 근육별 전수               →   Vlt2 (moment arm) + Vlt3 (FL/FV curve)
③ 통합 데이터 플로우 검증      →   Rajagopal 2016 보행 검증 (Kin + 부분 EMG)
```

CALM 검증 피라미드와의 매핑:

```
L0 — 수학 단위 테스트    ↔   Vlt3 (FL/FV curve 수치 정확도)
L1 — 모듈 데이터 흐름    ↔   Vlt2 (moment arm 전달 정확도)
L2 — 물리 엔진 연동      ↔   Vlt1 (forward kinematics 기하)
L3 — 현상론적 타당성     ↔   OpenSim 정성적 EMG 타이밍 비교
L4 — 데이터 정량 비교    ↔   Rajagopal 2016 보행 RMSE 기준
L5 — 환자 병리 패턴      ↔   해당 문헌 없음 (CALM 고유 계층)
```

## 6. CALM L4 기준에 대한 평가

현재 설계된 L4 합격 기준 (ver2.md):
- 키네마틱: RMSE < 8° (최소), < 8° (목표)
- GRF: RMSE < 0.15 BW
- EMG 타이밍: ±10% GC 이내

Rajagopal 2016 기준 대비:
- 키네마틱: Rajagopal < 1° RMSE (보행) vs CALM < 8° → **CALM이 관대함**
- 관절 모멘트: Rajagopal < 2% peak → CALM L4에는 모멘트 기준 없음 (추가 검토 여지)
- EMG: Rajagopal 정성적만 → **CALM이 더 엄격함**

결론: L4의 키네마틱 기준을 향후 강화할 여지 있음. 현재 기준은 "최초 통과" 수준으로 적절.
