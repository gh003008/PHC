## 테스트별 상세

### I01 — 토크 주입 방향 검증

- 시나리오: 초기 45°, +50Nm 토크를 L_Knee에 지속 주입
- 합격 기준: 각도가 증가(굴곡 방향) 하며 변화량 > 6°
- 결과: 45° → 145° (+100°)
- 의미: IsaacGym에서 양의 토크 = 양의 각도 변화(굴곡) 부호 규약 확인

### I02 — 토크 부호 규약 검증

- 시나리오: 초기 45°, 한 환경에 +50Nm, 다른 환경에 -50Nm 동시 주입
- 합격 기준: 두 환경의 최종 각도 차이 > 11°
- 결과: +50Nm → 145.0° (ROM 상한), -50Nm → -0.0° (ROM 하한). 차이 145°
- 의미: 양/음 토크 → 양/음 방향 운동이 정확히 대응

### I03 — 프로파일 분화 검증

- 시나리오: 초기 80°(1.4 rad), 신전 방향 초기 속도 -5 rad/s 부여, 5초간 관찰
- 합격 기준: 최종 각도 순서 `Spastic > Healthy > Flaccid`, 각 간격 > 5°
- 결과:
  - Spastic: 18.1° (강한 반사 + 높은 인대 강성 → 빠르게 감속 후 높은 위치 유지)
  - Healthy: 9.9° (정상 반사 → 중간 감속)
  - Flaccid: 3.3° (반사 없음 + 약한 근육 → 중력에 의해 가장 많이 떨어짐)
  - S-H 간격: 8.2° > 5° ✓
  - H-F 간격: 6.6° > 5° ✓
- 비고: BUG-01 수정 후 파라미터 재조정 수행 (별도 문서 참조: `260323_L2_프로파일_파라미터_조정_근거.md`)

### I04 — Stretch Reflex 차등 저항

- 시나리오: 초기 80°, 신전 방향 kick -3 rad/s, 0.5초간 bio-torque 측정
- 합격 기준: 초기 20step 평균 |bio-torque| 순서 `|Spastic| > |Healthy| > |Flaccid|`, S-H 차이 > 0.1 Nm
- 결과:
  - |Spastic|: 12.79 Nm (stretch_gain=15, threshold=0.01 → 강한 반사 토크)
  - |Healthy|: 9.26 Nm (stretch_gain=1.0, threshold=0.1 → 정상 반사)
  - |Flaccid|: 0.14 Nm (stretch_gain=0 → 반사 없음, 잔여는 수동력)
  - S-H 차이: 3.53 Nm > 0.1 Nm ✓
- 의미: stretch reflex가 프로파일별로 차등 작동하며, 물리 시뮬레이션에서 관측 가능한 수준의 토크 차이를 생성
