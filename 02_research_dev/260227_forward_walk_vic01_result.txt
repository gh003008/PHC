1. 실험 개요
- 실험 명: VIC01
- 모델 설명: 인간 모델 가변 임피던스 제어 (VIC) Stage 1 (Warm-up)

2. 정량적 평가 결과 (Test run with 512 envs)

- Mean Steps: 66.85
- 최대 보상: ~400 이상 관측 (안정 보행 시)
- 분석: 2N 차원의 확장된 액션 공간에서도 기존 HumanoidIm 모델(V4 등)과 유사하게 안정적으로 수렴함. 보행 안정성이 보존된 상태에서 임피던스 제어기가 정상 작동함을 확인.

3. 정성적 평가 결과 (Visual Verification)
- 실행 명령어: python phc/run.py --task HumanoidImVIC ... --num_envs 1 --test --no_virtual_display
- 보행 상태: 넘어짐 없이 전방으로 고르게 보행하며, Reference Motion과의 정렬 상태가 우수함.

- 결론: Stage 1 (고정 강성 보행) 목표 달성. 가변 임피던스 제어를 위한 신경망 구조와 물리적 제어 루프의 정합성이 검증됨.
- 향후 계획: vic_curriculum_stage를 2로 격상하여 가변 강성 학습(Stage 2) 시작. 대사 비용(Metabolic Cost) 최소화와 외란 극복 성능의 Trade-off 분석 예정.
