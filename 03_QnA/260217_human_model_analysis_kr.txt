PHC 인체 모델 분석 (웨어러블 로봇 통합용) 260217

1. PHC 모델 개요
PHC(Perceptual Humanoid Control)는 주로 SMPL 모델 기반의 인체 모델과 일부 휴머노이드 로봇 모델을 제공합니다.

제공 모델 리스트:
- SMPL: 표준 인체 모델. PHC에서 가장 널리 사용되며 24개의 관절로 구성됩니다.
- SMPL-H / SMPL-X: 손가락 움직임이나 얼굴 표정 등이 추가된 확장 모델입니다.
- Unitree H1 / G1: 휴머노이드 로봇 모델입니다. 인체 바이오메카닉스 시뮬레이션에는 SMPL이 더 적합합니다.

웨어러블 로봇 통합을 위한 추천:
- SMPL 모델을 추천합니다. 실제 수천 명의 인체 스캔 데이터를 기반으로 하기에 인체 치수와 기구학적 특성이 매우 사실적입니다.

2. 인체 모사 능력 분석
PHC의 SMPL 기반 모델이 실제 인간의 특성을 얼마나 잘 반영하는지 분석합니다.

기구학 및 외형:
- 성별과 체형 파라미터(betas)를 통해 모델이 동적으로 생성됩니다.
- 팔다리 길이, 관절 위치 등이 매우 사실적이며, 키가 크거나 마른 체형 등 다양한 피험자 시뮬레이션이 가능합니다.

동역학 (질량, 관성):
- 각 부위의 부피와 밀도를 계산하여 질량이 할당됩니다. 기본 밀도는 물과 유사한 1000 kg/m^3 수준입니다.
- PHC 내부 플래그를 통해 실제 인체의 부위별 질량 비율과 유사하게 보정하는 기능이 포함되어 있습니다.

관절 가동 범위 (ROM):
- Knees, Hips 등의 가동 범위가 설정되어 있으나, 일부는 학습의 편의를 위해 실제 인체보다 다소 넓게 설정되어 있습니다.

3. 강성 및 임피던스 정의
근육의 공동 수축(Co-contraction) 구현을 위해 중요한 요소입니다.

제어 방식:
- PHC는 PD 제어기를 사용하여 근육의 강성과 댐핑을 모사합니다.
- 현재는 고정된 Kp(강성), Kd(댐핑) 값을 사용합니다.
- 공동 수축을 구현하려면 정책(Policy)이 이 Kp, Kd 값을 실시간으로 가변 제어할 수 있도록 확장해야 합니다.

4. 웨어러블 로봇 통합 방안
- SMPL 모델 생성 스크립트에서 로봇의 링크를 인체 뼈대에 추가로 붙이는 방식이 가능합니다.
- 로봇 추가 시 전체 질량과 관성이 변하므로 제어기 안정성을 위해 PD 게인 조정이 필요할 수 있습니다.
