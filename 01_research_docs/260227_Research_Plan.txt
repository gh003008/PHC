[250227 인간-로봇 통합 시뮬레이션 및 가변 임피던스 연구 기획]

본 문서는 웨어러블 로봇 통합 시뮬레이션 구축을 위한 연구의 문제 정의와 단계별 기획안을 담고 있습니다.

1. 핵심 문제 정의 (Problem Definition)

기존 강화학습 기반의 인간 모델은 주로 고정된 제어 이득(Fixed PD Gains)을 사용하여 동작을 학습합니다. 
하지만 실제 인간은 외부 환경과의 상호작용이나 보행의 안정성을 위해 근육의 동시수축(Co-contraction)을 
통해 관절의 임피던스를 실시간으로 조절합니다.

특히 웨어러블 로봇은 인간의 신체에 직접 부착되어 힘을 전달하므로, 
인간과 로봇은 하나의 긴밀하게 결합된 시스템(Coupled System)이 됩니다. 
만약 시뮬레이션 속 인간 모델이 고정된 강성을 가진 로봇처럼 동작한다면, 
웨어러블 로봇의 보조 알고리즘이 실제 인간의 반응성이나 적응성을 제대로 반영하지 못하게 됩니다.

따라서 본 연구의 핵심 문제는 '인간의 가변 임피던스 특성을 반영한 디지털 휴먼 모델을 구축하고, 
이를 통해 로봇의 보조력이 인간의 동역학적 특성과 어떻게 상호작용하며 최적화되는가'를 정의하는 것입니다.

2. 연구의 단계별 로드맵

1단계: 인간 모델의 생체 모사 제어기 구축 (현재 단계)
- 가변 임피던스 제어(VIC)를 RL 정책에 통합하여 보행 안정성을 확보합니다.
- 물리적인 관절 제한(ROM)과 수동 PD 제어 루프를 통해 물리적 타당성을 높입니다.
- 결과물: 가변 강성을 스스로 조절하며 걷는 기초 인간 모델.

2단계: 외란 및 상호작용에 대한 견고성 검증
- 기존 고정 강성 모델(Baseline)과 VIC 모델을 수치적으로 비교합니다.
- 외부 밀치기(Push), 지면의 변화(Slope/Step) 등 외란 상황에서 VIC 모델이 동시수축을 통해 얼마나 더 안정적으로 균형을 잡는지 정량화합니다.
- 결과물: 가변 임피던스가 인간형 로봇의 강인성(Robustness)에 기여함을 증명.

3단계: 인간-로봇 통합 시스템 구축 (Coupled Simulation)
- Isaac Gym 내에 웨어러블 로봇(Exoskeleton) 모델을 도입하여 인간 모델과 결합합니다.
- 로봇의 무게, 링크 구조, 구동기 특성을 반영한 물리적 결합을 구현합니다.
- 결과물: 로봇을 착용한 상태에서의 인간 보행 시뮬레이션 환경.

4단계: 인간-로봇 공적응(Co-adaptation) 및 최적 보조 학습
- 로봇의 보조력과 인간의 임피던스 조절이 조화를 이루도록 학습합니다.
- 로봇의 보조가 강해질 때 인간이 스스로 강성을 낮추어 에너지를 아끼는지(Metabolic saving), 혹은 불안정한 상황에서 로봇과 인간이 협력하여 강성을 높이는지 분석합니다.
- 결과물: 인간의 근전도/대사 비용 특성을 고려한 최적의 로봇 보조 알고리즘 도출.

3. 기대 효과

이러한 접근 방식은 단순히 '잘 걷는 로봇'을 넘어서, '사람과 상호작용하는 로봇'을 평가할 수 있는 고충실도(High-fidelity) 테스트베드를 제공합니다. 이는 실제 임상 시험 전 시뮬레이션 상에서 로봇의 성능을 정밀하게 예측하고 최적화할 수 있는 강력한 도구가 될 것입니다.



[260227 가변 임피던스(VIC) 및 인간-로봇 통합 검증 전략]

본 문서는 제안된 가변 임피던스 기반 인간 모델이 얼마나 실제 인간과 유사한지, 
그리고 웨어러블 로봇 연구에 어떤 실질적인 가치를 제공하는지 증명하기 위한 검증 계획을 기술합니다.

1. 생체 모사 충실도 검증 (Bio-fidelity Verification)
인간 모델이 실제 인간의 물리적 특성을 얼마나 잘 모사하는지 정량화합니다.

- 임피던스 범위 비교: 학습된 강성(Stiffness) 값의 범위를 실제 인체 실험 데이터
(예: 보행 중 발목 강성 변화 문헌값)와 비교하여 0.5배~2.0배 변화가 생체 역학적으로 
타당한 범위 내에 있는지 확인합니다.
- 동시수축 지수 (Co-contraction Index, CCI): 근육 모델을 직접 쓰지 않더라도, 
토크 기반으로 환산된 CCI가 보행 주기의 특정 구간(Heel strike 등)에서 실제 인간처럼 
증가하는지 패턴을 분석합니다.
- 대사 비용 상관관계: 가변 임피던스를 사용했을 때 소모되는 전체 에너지(Metabolic cost)가
 고정 강성 모델 대비 실제 인간의 보행 효율 패턴과 얼마나 유사하게 수렴하는지 측정합니다.

2. 제어 견고성 검증 (Robustness Verification)
가변 임피던스가 '단순히 잘 걷는 것'을 넘어 '외란에 얼마나 강한지' 입증합니다.

- 외력 섭동 테스트 (Push Recovery Test): 보행 중인 모델의 몸통(Torso)에 다양한 크기의 충격량(Impulse)을 가하고, 넘어지지 않고 복구되는 성공률을 Baseline과 비교합니다. 이때 VIC 모델이 충격 순간 강성을 높여 대응하는지 확인합니다.
- 비정형 지연 테스트: 경사로(Slope), 요철(Uneven terrain), 미끄러운 지면 등에서 보행의 안정성(Center of Mass의 흔들림 정도)을 정량적으로 평가합니다.
- 지표: 넘어짐 비율 (Fall rate), 보행 대칭성 (Symmetry), 복구 시간 (Recovery time).

3. 인간-로봇 상호 적응 검증 (Co-adaptation Verification)
웨어러블 로봇 연구에서 가장 중요한 '로봇 보조에 따른 인간의 반응'을 검증합니다.

- 보조력 적응 테스트: 웨어러블 로봇이 보조력(Assistive torque)을 제공할 때,
 인간 모델이 에너지 효율을 위해 자신의 관절 강성을 스스로 낮추는지(Stiffness reduction) 확인합니다.
- 저항력 적응 테스트: 반대로 로봇이 보행을 방해할 때, 인간 모델이 안정성을 위해 강성을 높이며 저항하는지
 분석합니다. 
- 지표: 로봇 보조 유무에 따른 인간 가변 강성의 변화량, 로봇 착용 시 인간의 대사 비용 절감률
 (Metabolic saving rate).

4. 정량적 검증 시나리오 요약

- 테스트 1: (고정 강성 모델) vs (VIC 모델) - 외력 100N 적용 시 복구 성공률 비교.
- 테스트 2: 보행 주기별 learned stiffness plot 시각화 및 문헌값(Human gait data) 오버레이 비교.
- 테스트 3: 웨어러블 로봇 착용 전/후 인간 관절의 임피던스 변화 맵 작성.

이러한 다각도의 검증을 통해 본 연구가 '단순한 제어 시뮬레이션'이 아닌
 '실제 로봇 개발을 위한 고충실도 시뮬레이터'임을 증명합니다.
